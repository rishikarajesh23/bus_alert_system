# RSET Bus Alert System 

## Overview

This project is a **Flask-based web application** designed to provide users with **real-time bus schedule updates**. It ensures that commuters stay informed about bus timings and important alerts through an intuitive interface and automated notification system.

---

## Features

* Real-time bus schedule updates
* Automated email notifications for alerts
* Firebase integration for data management
* Interactive and user-friendly frontend
* Responsive design for better usability

---

## Problem Statement

Commuters often face difficulties in accessing **accurate and up-to-date bus schedules**, leading to delays and inconvenience.

---

## Solution

This application provides:

* Live schedule updates
* Instant alerts via email
* A centralized and easy-to-use platform

Helping users plan their travel efficiently and avoid unnecessary waiting time.

---

## Tech Stack

### Software Components

* **Backend**: Flask (Python)
* **Frontend**: HTML, CSS, JavaScript
* **Database & Services**: Firebase
* **Notifications**: Firebase Email Alerts

---

## Workflow

1. User accesses the web application
2. Bus schedule data is fetched from Firebase
3. Real-time updates are displayed on the UI
4. Alerts/changes trigger automated email notifications
5. User receives timely updates and notifications
